Goober Dash Level Modding!

A bunch of silly little scripts to generate levels for Goober Dash. Please don't rely on these tools being consistent or accurate as they may lose or break details of their outputs.

Grid

Create a grid with a specified number of rows and columns.

Pixel Art

Upload an image to convert it into pixel art.

Impossible Gravity

Generate a Gravity Field with strength over 5.

Scale Level

Scale up an entire level.

Mirror Level

Mirror a level horizontally and/or vertically.

Speed Up Animations

Speed up all animations in a level.

Randomize objects

Randomly swap out all the object types in a level.

Apply Gravity

Apply an impossible gravity strength to all fields in a level.

Level Details

View details and stats about the content of a level.

Help

How to use tools?

Most tools will output a .json file that you can open in game by dragging it onto the browser window while in the Goober Dash editor

Can't find a tool?

Use the nav bar at the top of the page to quickly find each tool. If a tool is missing from the page, you may just need to hard refresh the page to get the newest version by pressing ctrl + shift + R

Want another tool?

Feel free to message me on discord @.index and recommend new tool ideas or additions to current tools and I will likely add them to the page
